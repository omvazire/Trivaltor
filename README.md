# 🌍 TRIVALTOR GROUP — Export Business Website

A production-ready **MERN Stack** web application developed for a real export business to showcase products, generate leads, and manage enquiries through a secure admin dashboard.

> **Note:** This project was developed independently for a real client as a freelance software developer and served as my practical industry experience in place of a traditional internship.

🌐 **Live Website:** https://www.trivaltorgoc.com

---

## ✨ Features

### Public Website

- Modern responsive UI
- Product catalogue with categories
- Product detail pages
- Company profile
- Contact page
- WhatsApp integration
- Mobile-first design
- Smooth animations
- SEO-friendly structure

### Lead Management

- Popup lead collection
- Product enquiry forms
- Visitor tracking
- Lead validation
- Automatic MongoDB storage
- Conversion tracking

### Admin Dashboard

- Secure JWT Authentication
- Protected admin routes
- Dashboard analytics
- Visitor statistics
- Popup lead management
- Enquiry management
- Contact status updates
- CSV export
- Pagination

### Analytics

- Total visitors
